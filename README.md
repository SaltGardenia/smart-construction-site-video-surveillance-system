# 智慧工地

## 简介

本项目是基于 pyqt5 + OpenCV 实现视频播放、摄像头监控、云端检测等功能，适用于智慧工地的视频管理和监控。

## 主要功能

- 视频播放和暂停
- 摄像头监控（开启和关闭）
- 云检测（对接华为云平台）
- 视频文件类别的展示和操作（点击播放和自动加载视频列表）

## 依赖环境

- python3.7+
- pyqt5
- opencv-python

## 目录结构

- check_videos：已经检测完的视频
- image：项目的背景图片和图标
- video：录制的视频
- views：页面代码
- main.py：程序入口

## 运行方式

1、安装依赖：pyqt5、opencv

2、运行程序：main.py
